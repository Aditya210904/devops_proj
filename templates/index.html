About the Project

This project demonstrates a fully automated DevOps pipeline built for local testing and deployment. It connects GitHub Actions, Docker, and Terraform for continuous integration and delivery.

Each commit triggers unit tests, builds a Docker image, and deploys it automatically to a local Docker Swarm environment — ensuring reliable, hands-free deployment.
